# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting significant design choices in Backup Sentinel.

Each ADR follows the format:
- **Context** — What is the situation that calls for a decision?
- **Decision** — What is the chosen approach?
- **Consequences** — What are the trade-offs, both positive and negative?
